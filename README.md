## 👋 关于

做项目时我更在意这几件事：

- **能跑**：本地 / Docker 一键起来，而不是只有截图  
- **能验**：接口有测试、关键路径有回归  
- **能看**：日志、健康检查、监控，出问题知道从哪查  
- **能复用**：配置、脚本、文档写清楚，方便下次接着改  

日常在写 Python 服务、自动化测试、运维小实验，以及把 LLM 接到实际工作流里的小工具。
